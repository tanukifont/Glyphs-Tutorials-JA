# GlyphsにWindows版がない理由

GlyphsにWindows版がないのはなぜか、不思議に思ったことはありませんか？これがその理由です。

私たちは時々、GlyphsのWindows版を求めるリクエストを受け取ります。私たちはクロスプラットフォーム化を基本的に否定しているわけではありませんが、現時点ではそのような試みは経済的に実現不可能だと考えています。たとえすべてのリクエストがライセンスの販売に繋がったとしても（これは非常に楽観的な仮定ですが）、その売上では開発とサポートにかかる追加コストを賄うことはできません。そして悲しいことに、到底及びません。

さらに悪いことに、これらの数少ないリクエストのほとんどは、Glyphsのフルバージョンではなく、[Glyphs Mini](/glyphs-mini)に関するものです。つまり、年間数百ドルの期待収益のために、数十万ドルもの投資をしなければならないということになります。それは私たちにはできません。しかし、もしWindows市場にその投資価値があると信じ、そのようなリスクを負うことを厭わない投資家の方がいらっしゃいましたら、ぜひご連絡ください。ただし、この方法ではおそらく大金を失うことになるでしょう、ということは申し上げておきます。

#### 書体デザインはMacのビジネスです

現時点で私たちが言えるのは、書体デザインはほぼMacプラットフォーム上で独占的に行われているように見える、ということです。私たちが訪れるすべてのカンファレンスで、ワークショップを行うすべての学校で、そして仲間の書体デザイナーたちと交わす多くの個人的な会話の中で、そのことを目の当たりにします。言い換えれば、Windows版を実際に制作、管理、維持するよりも、Windows版を求めるすべての人にMacを配る方が、私たちにとっては安上がりだということです。もちろん、私たちはそれもしませんが。

状況が変わるまでは、これらのリクエストをお断りせざるを得ません。申し訳ありません。
